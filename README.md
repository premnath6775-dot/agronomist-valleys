# AgronoMist Valley's

B2B commercial wholesale vegetable supplier based in Malaysia. Distributing high-quality produce to top-tier restaurants and grocers.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `npm run dev`
